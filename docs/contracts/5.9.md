# Task 5.9: Duplicate detection — filename + size cross-provider comparison

## Overview
API endpoint to detect duplicate files across multiple storage providers based on filename and size comparison.

## Endpoint
`GET /api/cleanup/duplicates`
`POST /api/cleanup/duplicates`

## Request Shape

### GET Request
Query parameters:
- `minSize` (optional, number): Minimum file size to consider for duplicate detection (default: 0)
- `maxSize` (optional, number): Maximum file size to consider for duplicate detection (default: Infinity) 
- `includeTrash` (optional, boolean): Whether to include files in trash/recycle bin (default: false)
- `recursive` (optional, boolean): Whether to scan recursively through folders (default: true)
- `providers` (optional, string): Comma-separated list of provider IDs to scan (default: all connected providers)

### POST Request
JSON body:
```json
{
  "minSize": 0,
  "maxSize": 1000000000,
  "includeTrash": false,
  "recursive": true,
  "providers": ["google", "dropbox"]
}
```

## Response Shape
```json
{
  "success": true,
  "count": 5,
  "duplicates": [
    {
      "signature": "document.pdf|1024",
      "fileName": "document.pdf",
      "fileSize": 1024,
      "files": [
        {
          "id": "file123",
          "providerId": "google",
          "name": "document.pdf",
          "size": 1024,
          "parentId": "folder456",
          "path": "/Documents/document.pdf",
          "mimeType": "application/pdf",
          "modifiedAt": "2023-01-01T10:00:00Z",
          "webUrl": "https://drive.google.com/file/d/file123/view"
        }
      ]
    }
  ]
}
```

## Error Response Shape
```json
{
  "success": false,
  "error": "Error message",
  "code": "ERROR_CODE"
}
```

## Error Codes
- `VALIDATION_FAILED`: Invalid request parameters/body
- `INTERNAL_ERROR`: Internal server error during duplicate detection
- `FEATURE_NOT_IMPLEMENTED`: Backend functionality not yet implemented

## Edge Cases
- Provider authentication failures: Skip provider and continue with others
- Empty provider list: Return empty duplicates array
- No duplicates found: Return empty duplicates array with count 0
- File listing failures: Log error and continue with other providers
- Mixed provider types: Compare files across all supported providers

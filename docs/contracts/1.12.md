# Contract: 1.12
## Endpoint URL or File Path
- Endpoint URL (response wrapper contract): `/api/remotes/audits/upload-uuid-injection`
- `api/remotes/uploadUuidAudit.js`
- `lib/uploads/uuidInjectionAudit.ts`
- `lib/providers/types.ts`

## Request / Response Shape
- Scanner entrypoint:
  - `buildUploadUuidAuditReport(repoRoot?: string) -> UploadUuidAuditReport`
  - `repoRoot` defaults to current working directory and is used to resolve audit target files.
- Response wrapper:
  - `createUploadUuidAuditResponse(report) -> { ok: true, endpoint, data } | { ok: false, endpoint, error }`

### `UploadUuidAuditReport`
```ts
{
  gate: 'UUID-1',
  generatedAt: string, // ISO-8601
  points: UploadUuidInjectionPoint[],
  summary: {
    scannedFiles: number,
    scannedLines: number,
    injectionPoints: number,
    uploadPathInjectionPoints: number,
    hasBlockingUploadPathInjection: boolean
  }
}
```

### `UploadUuidInjectionPoint`
```ts
{
  id: string,
  filePath: string,
  line: number,
  expression: string,
  snippet: string,
  category: 'request_tracking' | 'transfer_tracking' | 'resumable_session' | 'remote_config' | 'oauth_state' | 'qa_mock' | 'upload_path' | 'unknown',
  affectsUploadPath: boolean,
  notes: string
}
```

## Error Codes
- `UUID_AUDIT_TARGET_MISSING`: an expected audit target file does not exist.
- `UUID_AUDIT_FILE_READ_FAILED`: audit scanner cannot read one of the configured files.
- `UUID_AUDIT_REPORT_INVALID`: response wrapper received malformed report data.

## Edge Cases
- Missing target files are represented as findings (`expression: "MISSING_FILE"`) instead of silently passing.
- UUID-like identifiers used only for request tracing, queue job IDs, or OAuth state are still reported but flagged with `affectsUploadPath: false`.
- Upload route path mutator scan is strict to path-building lines (`relativePath`, `originalname`, `filename`, `diskPath`) to avoid false positives from unrelated token generation code.
- `hasBlockingUploadPathInjection` is `true` only when UUID generation is detected in persisted upload path composition logic.

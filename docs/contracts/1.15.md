# Contract: 1.15
## Endpoint or File Path
- `scripts/migrate-files-no-uuid.js`

## Request / Response Shape
- Production/staging execution path:
  - `node scripts/migrate-files-no-uuid.js`
- Behavior per UUID-prefixed file:
  - move disk paths from UUID-prefixed to clean path
  - update DB path (`UPDATE files SET path = $1 WHERE id = $2`)
  - record change in migration log
- Final console summary reports totals for checked, migrated, and errors.

## Error Codes
- Hard failure exits process with code `1` on top-level migration error.
- Individual file migration errors are captured and reported without immediate exit.

## Edge Cases
- Handles both local cache path and pool path if either exists.
- Removes redundant UUID-prefixed records when content hashes match.
- Preserves unresolved conflicts (same target path, different hash) for manual follow-up.

## Example Payload
```bash
DATABASE_URL=<postgres-url> node scripts/migrate-files-no-uuid.js
```

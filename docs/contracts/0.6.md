# Contract: 0.6 - Singleton Refresh Promise Pattern

## Overview

The refresh guard pattern ensures only one token refresh executes at a time, preventing concurrent refresh race conditions.

## File Path

- **Module**: `lib/auth/refreshGuard.ts`
- **Tests**: `tests/unit/refreshGuard.test.ts`

## API

### `initRefreshGuard(refreshFn: RefreshFn): RefreshGuard`

Initialize the singleton refresh guard with a refresh function.

- **Parameters**: `refreshFn` - Async function that returns a new access token
- **Returns**: `RefreshGuard` instance

### `getRefreshGuard(): RefreshGuard`

Get the singleton refresh guard instance.

- **Returns**: `RefreshGuard` instance
- **Throws**: Error if not initialized

### `RefreshGuard.getToken(): Promise<string>`

Get a valid token, waiting for any ongoing refresh.

- **Returns**: Promise resolving to access token

### `RefreshGuard.isRefreshing(): boolean`

Check if a refresh is currently in progress.

- **Returns**: `true` if refresh in progress

### `RefreshGuard.reset(): void`

Manually reset the guard state (clears any ongoing promise).

## Request/Response Shape

This is a client-side module, not an HTTP endpoint.

```typescript
// Initialization
type RefreshFn = () => Promise<string>;
initRefreshGuard(refreshFn: RefreshFn): RefreshGuard

// Usage
const guard = getRefreshGuard();
const token = await guard.getToken();
```

## Error Codes

| Error | Description |
|-------|-------------|
| `RefreshGuard not initialized` | Thrown when `getRefreshGuard()` called before `initRefreshGuard()` |

## Edge Cases

1. **Concurrent requests**: All concurrent `getToken()` calls receive the same promise
2. **Refresh failure**: Promise clears after failure, allowing retry
3. **Multiple 401s**: Only one refresh triggers, others wait

## Example Payload

```typescript
// Initialize
initRefreshGuard(async () => {
  const res = await fetch('/api/auth/refresh', {
    method: 'POST',
    credentials: 'include',
  });
  const { accessToken } = await res.json();
  return accessToken;
});

// Use
async function authenticatedRequest(url: string) {
  const token = await getRefreshGuard().getToken();
  return fetch(url, {
    headers: { Authorization: `Bearer ${token}` },
    credentials: 'include',
  });
}
```

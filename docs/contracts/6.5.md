# Trash & Versioning API Contract (6.5)

## Trash Operations

### List Trash
**Endpoint:** `GET /trash`
**Request:**
- `cursor` (optional, string): Pagination cursor
- `pageSize` (optional, number): Number of items to return

**Response Shape:**
```json
{
  "files": [
    {
      "id": "file_id",
      "name": "filename.txt",
      "providerId": "google",
      "size": 1024,
      "isFolder": false,
      "trashedAt": "2023-01-01T00:00:00Z",
      "originalPath": "/path/to/file"
    }
  ],
  "nextCursor": "next_page_token",
  "hasMore": true
}
```

### Restore File
**Endpoint:** `POST /trash/{id}/restore`
**Request:** Empty
**Response Shape:** Empty (204 No Content)

### Permanently Delete File
**Endpoint:** `DELETE /trash/{id}`
**Request:** Empty
**Response Shape:** Empty (204 No Content)

### Empty Trash
**Endpoint:** `POST /trash/empty`
**Request:** Empty
**Response Shape:** Empty (204 No Content)

## Versioning Operations

### List File Versions
**Endpoint:** `GET /files/{fileId}/versions`
**Request:** Empty
**Response Shape:**
```json
{
  "versions": [
    {
      "id": "version_id",
      "modifiedAt": "2023-01-01T00:00:00Z",
      "size": 1024,
      "author": "User Name"
    }
  ]
}
```

### Restore File Version
**Endpoint:** `POST /files/{fileId}/versions/{versionId}/restore`
**Request:** Empty
**Response Shape:** Empty (204 No Content)

## Error Codes
- `UNAUTHORIZED` (401): Missing or invalid auth token.
- `NOT_FOUND` (404): File or version not found.
- `VALIDATION_FAILED` (400): Invalid request parameters or operation not supported by provider.
- `PROVIDER_UNAVAILABLE` (503): Upstream provider error.

## Edge Cases
- **Restoring to original path:** If the original parent folder was also deleted, the file may be restored to the root or an available parent depending on the provider.
- **Provider support:** Some operations like "Empty Trash" or "Restore Version" are not supported natively by all providers and may return an error.
- **Pagination:** Large trash bins will require multiple calls using the `nextCursor`.

# Contract: 1.14
## Endpoint or File Path
- `scripts/migrate-files-no-uuid.js`

## Request / Response Shape
- CLI modes:
  - `node scripts/migrate-files-no-uuid.js --dry-run`
  - `node scripts/migrate-files-no-uuid.js` (apply)
- Reads DB rows:
  - `SELECT id, user_id, path, status, hash FROM files ...`
- Writes migration log JSON:
  - `{ timestamp, totalFiles, migratedCount, errorCount, changes[] }`

## Error Codes
- Process exits with code `1` when:
  - `DATABASE_URL` is missing
  - uncaught migration failure occurs
- Per-file migration errors are counted and logged with `errorCount`.

## Edge Cases
- Duplicate destination path with same hash: marks redundant row as deleted and removes redundant file.
- Duplicate destination path with different hash: skipped for manual resolution.
- Cleans empty UUID directory segments in both local and pool paths.

## Example Payload
```bash
node scripts/migrate-files-no-uuid.js --dry-run
```
